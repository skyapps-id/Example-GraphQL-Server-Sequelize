# Graphql & Sequelize

### Dependency 
- Apollo Server
- Sequelize
- Mysql2

### Installation Manual

1. Clone this project.
   
    ```sh
    $ git clone https://github.com/skyapps-id/GraphQL-Server-Sequelize.git
    $ cd GraphQL-Server-Sequelize/
    ```

2. Install Dependency
   
    ```sh
    $ npm install 
    ```

3. Config Database
   
    ```sh
    $ vi config/config.json 
    ```

4. Run Project
   
    ```sh
    $ npm run start
    ```
